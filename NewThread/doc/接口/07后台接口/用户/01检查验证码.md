#### /admin/c.do
---------------------------

#### 接口说明
- 此接口可提供给AJAX校验验证码,登录接口会再一次校验,可以无限校验,不过时间戳的Cookie有效期为1分钟

#### URL
- 前缀/c.do

>前缀的说明请看README.md

#### HTTP请求方式
- GET

#### 是否需要登录
- 否

#### 访问授权限制
- 访问级别：普通接口
- 频次限制：否

#### 参数类型
- 普通URL参数

#### 请求参数
|参数字段|字段类型|字段说明|
|:----------:|:--------:|:---------:|
| c  |  String | 验证码 |

#### 注意事项
- 无

#### 返回结果
|返回字段|字段类型|字段说明|
|:----------:|:--------:|:---------:|
| 0或1  | int | 1为验证码正确,0为错误 |


#### 返回字段说明
- 无
