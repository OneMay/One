#### /admin/gpr.do
---------------------------

#### 接口说明
- 获取本用户所有权限

#### URL
- 前缀/admin/gpr.do

>前缀的说明请看README.md

#### HTTP请求方式
- GET

#### 是否需要登录
- 是

#### 访问授权限制
- 访问级别：普通接口
- 频次限制：否

#### 参数类型
- 普通URL参数

#### 请求参数

#### 注意事项
- 无

#### 返回结果
|返回字段|字段类型|字段说明|
|:----------:|:--------:|:---------:|
| 数组 | list | 包含权限的数组 |

#### 返回结果
- 正确：

        [
          "edit",
          "delete",
          "update",
          "query"
        ]

- 错误：

	{
		"code":"E01",
		"message":"无权进行此项操作!",
		"content":null
	}

#### 返回字段说明
- 无
